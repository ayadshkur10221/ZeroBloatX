## 🚀 How to Run It (Super Easy 3-Step Guide)

You don't need to be a tech genius to use this. Just follow these steps:

### 1️⃣ Turn Off Tamper Protection (Crucial!)
Windows Defender will try to block the script to protect itself. Before running the tool, you *must* temporarily pause it:
*   Open your Start Menu, search for **Windows Security**, and open it.
*   Go to **Virus & threat protection** -> click **Manage settings**.
*   Toggle **Tamper Protection** to **OFF**.

### 2️⃣ Run the Tool
*   Download and extract the **ZeroBloatX** folder.
*   **Right-click** on `ZeroBloatX.bat` and select ⭐ **Run as Administrator** ⭐.
*   Press **Y** when it asks to apply the tweaks, and let it do its magic!

### 3️⃣ Restart Your PC
Once the script says `[✓] Done!`, close the window and **restart your computer** to fully apply the massive performance boost!
